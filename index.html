DaySnap CLI

One Day One Snap

功能丰富

对 Babel、TypeScript、ESLint、PostCSS、PWA、单元测试和 End-to-end 测试提供开箱即用的支持。

易于扩展

它的插件系统可以让社区根据常见需求构建和共享可复用的解决方案。

无需 Eject

DaySnap CLI 完全是可配置的，无需 eject。这样你的项目就可以长期保持更新了。

# 起步

安装：

npm install -g @daysnap/cli
# OR
yarn global add @daysnap/cli

查看命令：

dsc
# OR
dsc -h
